# Timescale Hotkey

- Allows you to speed up or slow down time by pressing a dedicated configuranble key
- You can add additional key in case you want separate buttons for slowing down or speeding up time

## Installation

Install [MelonLoader](https://github.com/LavaGang/MelonLoader) if it's your first mod

Get the latest [release](https://github.com/HAHAYOUDEAD/TimescaleHotkey/releases) and put the .dll into `.../Mods/`

Requires [ModSettings](https://github.com/DigitalzombieTLD/ModSettings/) (also just drop its .dll into `.../Mods/`)
